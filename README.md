# Weather Here

A React-based weather application that provides current weather information for cities worldwide.

## Features

- Search for weather by city name
- Display current temperature, weather conditions, and activity tips
- Toggle between Celsius and Fahrenheit
- Add cities to favorites
- Responsive design with smooth animations

## Installation

1. Clone the repository:
   ```
   git clone https://github.com/shreyansh6726/Weather-here.git
   ```

2. Navigate to the project directory:
   ```
   cd Weather-here
   ```

3. Install dependencies:
   ```
   npm install
   ```

## Usage

1. Start the development server:
   ```
   npm start
   ```

2. Open your browser and go to `http://localhost:3000`

3. Search for a city or select from the predefined list to view the weather.

## Technologies Used

- React
